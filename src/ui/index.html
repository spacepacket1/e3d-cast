Ethereum E3D vs Base wE3D

Ethereum E3D (ticker: E3D) is the native token on Ethereum mainnet. Send E3D directly from your Ethereum wallet to purchase Cast credits in one transaction.

Base wE3D is wrapped E3D bridged to Base. It settles faster and costs less gas, making it the preferred option for agent automation and frequent API payments.

Both tokens share the same credit rate: 1 E3D = 1,000 credits. Credits are non-custodial — they are spent per job at submission time.
